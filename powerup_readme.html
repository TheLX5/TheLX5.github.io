Custom Powerups Readme

This patch lets the user insert custom powerups and edit existing powerups. It also provides several resources to make powerups and other stuff easier.

Feel free to translate this file to other languages. Report it to me so I can make an online version of the readme in said language. The same goes if you want your powerup to be included in a future version of the powerup pack.

Note that this is a huge compilation of ASM hacks, this means that there are major incompatibilities with other resources. It's up to the author of said resources to fix these issues, not me.
