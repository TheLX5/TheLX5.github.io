Setting up the new graphics


The player graphics format is the same as they are in Ladida's 32x32 player tilemap kit patch, each frame is a 32x32 square in the bin file. I suggest using a file that is already included with the patch to know which poses are which.
To include your new graphics, you have to put your bin file in the powerups_files/graphics folder. Then open powerup_gfx.asm which is in powerups_files folder and add your graphics file there too using the macros you see in the file.
Macro for bin files with a size <= 32KiB:

%insert_gfx(filename,id)

Macro for graphic files with a size between 32KiB and 64KiB:

%insert_gfx_big(filename,id)

Basic code for Player ExGFX change


Now that you included your own custom graphics, it's time to make use of them.

Change Mario GFX:

  LDA #$01
  STA !gfx_bypass_flag
  LDA #$XX      ;There goes the ID of your graphic file.
  STA !gfx_bypass_num

The code is rather simple, with a bit of work you can have a whole new set of different suits for Mario!
