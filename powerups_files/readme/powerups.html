Hammer Suit (Super Mario Bros. 3)


Description
The iconic Hammer Suit from Super Mario Bros. 3 is available in this powerup pack. With it, Mario can throw hammers that describes an arc movement. Their horizontal speed is dependent on how fast is Mario going. The hammers also are super powerful, you can defeat enemies with a single hammer unless, of course, they are immune!


Controls
  • X/Y: Shoot hammer.
  • A: Shoot hammers while spin-jumping.

Customization
The powerup is highly customizable, To modify it, you can go to /powerups_files/powerup_defs folder and open powerup_04.asm. Please read through it first before changing anything there!
You can also change the projectile graphics in the extended sprites folder.
There are a few blocks that work with this powerup inside of the GPS folder.

Super Leaf (Super Mario Bros. 3)


Description

Another iconic powerup! The Super Leaf from Super Mario Bros. 3 is fully functional. This powerup lets Mario fly for a short period of time, it also gives him a more accurate Tail attack and animation.



Controls

  • X/Y: Spin attack.
  • X/Y in the air: Spin attack.
  • Run + B: Start flying.
  • While flying, tap B: Go higher.
  • While descending, tap B: Descend slowly.

Customization

The powerup is highly customizable, To modify it, you can go to /powerups_files/powerup_defs folder and open powerup_06.asm. Please read through it first before changing anything there!

Tanooki Suit (Super Mario Bros. 3)


Description

The Tanooki Suit powerup makes an appearance here, it lets Mario do the same things as a Super Leaf, but with the exception that he also can transform himself into a statue, becoming invincible and crushint enemies beneath him!



Controls
  • X/Y: Spin attack.
  • X/Y in the air: Spin attack.
  • Run + B: Start flying.
  • While flying, B: Go higher.
  • While descending, B: Descend slowly.
  • D-Pad down + tap X/Y: Transform into a statue.
  • In the air, D-Pad down + tap X/Y: Statue drop.
  • While being in statue form, D-Pad down: Keep statue form.
  • While being in statue form, release all buttons: Revert statue form.

Customization

The powerup is highly customizable, To modify it, you can go to /powerups_files/powerup_defs folder and open powerup_07.asm. Please read through it first before changing anything there!
There are a few blocks that work with this powerup inside of the GPS folder.
