How can I remove powerups that I have no plans to use?


You can't.
It's not on my plans to offer such option, at most you can trim down to 1 byte each graphics file you don't want (don't remove them, the patch will whine about it) and call it a day.
I'm not releasing powerups in an add-on fashion either, that sounds like a lot of effort and I don't wanna waste time on such things.

Is it possible to rearrange the way the powerups are listed?


This is possible if you know which files are you supposed to change. It is not recommended to do it if you have little to no experience with ASM, you might get confused.
What you have to do is modify the necessary files in the following folders:
  1. powerup_main_code
  2. powerup_image_code
  3. powerup_interaction_code
  4. powerup_defs
  5. powerup_misc_data
You have to modify a lot of things and it won't be worth the time spent on it, but if you wanna do it, go ahead. Just don't expect much support on tweaking the core files of the patch and expect everything to magically work.

Are custom status bars compatible with this?


No, but I do have plans to update my own custom Sprite Status Bar patch to be compatible with this. I'd like these other patches floating around to support this patch natively rather than offering separate downloads in the release thread.
